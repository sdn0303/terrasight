## 4. J-SHIS 調査結果

### 4.1 概要

[J-SHIS（地震ハザードステーション）](https://www.j-shis.bosai.go.jp/) は防災科研が運営する地震ハザード情報提供サービス。

### 4.2 利用可能 API（認証不要・無料）

| API | エンドポイント | 返却形式 | 不動産投資との関連 |
|-----|-------------|---------|-----------------|
| **地震ハザード情報** | `/map/api/pshm/{Y2024}/AVR/TTL_MTTL/meshinfo.geojson?position={lng},{lat}&epsg=4326` | GeoJSON (250mメッシュ) | **◎ 高** — 地震リスクスコアの中核 |
| **表層地盤情報** | `/map/api/sstrct/V3/meshinfo.geojson?position={lng},{lat}&epsg=4326` | GeoJSON (250mメッシュ) | **◎ 高** — AVS30（軟弱地盤≒液状化間接指標） |
| **地すべり地形** | `/map/api/landslide/isContaining.json?position={lng},{lat}&epsg=4326` | JSON (boolean判定) | **○ 中** — steep_slope 補完 |
| **長期間平均ハザード** | `/map/api/avghzd/V1/meshinfo.geojson?meshcode={code}` | GeoJSON (250mメッシュ) | △ 低 — 地震ハザードと重複 |
| 地震活動モデル(断層) | `/map/api/pshm/{Y2024}/AVR/{eqcode}/fltinfo.geojson` | GeoJSON | △ 低 — 可視化用 |

### 4.3 主要 API パラメータ

#### 地震ハザード情報 (pshm)

| パラメータ | 値 | 説明 |
|-----------|-----|------|
| `version` | Y2008〜Y2024 | 予測地図バージョン |
| `case` | AVR / MAX | 平均ケース / 最大ケース |
| `eqcode` | TTL_MTTL | 全地震（主部断層＋その他） |
| `format` | geojson / gml | レスポンス形式 |
| `position` | {lng},{lat} | 経緯度指定（meshcodeと排他） |
| `epsg` | 4326 / 4612 / 4301 | 座標参照系 |

#### 返却属性（28種）

- **30年確率**: 震度5弱/5強/6弱/6強以上となる確率、3%/6%超過確率時の震度・最大速度
- **50年確率**: 2%/5%/10%/39%超過確率時の震度・最大速度

### 4.4 利用規約

| 項目 | 内容 |
|------|------|
| APIキー | **不要** |
| 利用料 | **無料** |
| 編集・加工 | ✅ 自由 |
| ファイルそのままの複製・再配布 | ❌ 禁止 |
| 商用販売 | 要事前承認 |
| 出典表示 | **必須**（「J-SHIS 地震ハザードステーション」） |
| レート制限 | あり（具体値未公開、HTTP 403で通知） |

### 4.5 データ取得方式の選択

| 方式 | メリット | デメリット | 推奨 |
|------|---------|----------|------|
| **API動的取得** | 常に最新、メッシュ単位で必要分だけ | レート制限あり、レイテンシ追加 | ✅ スコア算出時のリアルタイム取得 |
| **Shapefile一括** | オフライン利用可、高速 | 静的、全国分は大容量 | △ PostGIS投入してキャッシュ |

→ **推奨**: Rust バックエンドから API コール → SQLite キャッシュ（24h TTL）→ スコア算出に利用。`P-Y2024-PRM-SHAPE.zip` はフォールバック用。

### 4.6 液状化データについて

**J-SHIS は「液状化」データを提供していません。**

スクリーンショットのデータツリーにある項目は全て地震動・地盤構造系:

- 確率論的地震動予測地図 → 地震の揺れの確率
- 条件付超過確率 → 特定条件下の揺れ
- 震源断層を特定した地震動予測地図 → 特定断層の影響
- 地盤情報（表層地盤/深部地盤） → 地盤の物理特性
- 曝露人口 → 被災想定人口

ただし **表層地盤の AVS30（地表30m平均S波速度）** は液状化リスクの間接指標として利用可能。AVS30 が低い地域 = 軟弱地盤 = 液状化リスク高。

### 5.2 PL分布図の利用手順

```bash
# 1. Shapefile → GeoJSON 変換
ogr2ogr -f GeoJSON -t_srs EPSG:4326 liquefaction.geojson PL分布図.shp

# 2. PostGIS 投入
ogr2ogr -f "PostgreSQL" PG:"host=localhost dbname=realestate" liquefaction.geojson \
  -nln liquefaction -overwrite
```

## 6. データパイプライン方針

### 6.1 静的データ（ファイルダウンロード → PostGIS）

```
data/*.zip → unzip → ogr2ogr (GML/SHP→GeoJSON) → PostGIS INSERT
                                                     ↓
                                              東京23区 bbox フィルタ
```
